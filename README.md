# OnekoSharp
A simple Oneko app made in C#

## Features

- All Oneko's original interactions (Walking, idle, scratching, sleeping and waking up)
- Put Oneko in a box
- Customizable settings
  - Change Oneko size and speed
  - Set a global keybind to put Oneko in a box
